# Code of Conduct

This project expects respectful, professional collaboration.

Participants should:

- Use welcoming and inclusive language.
- Respect different viewpoints and experience levels.
- Focus criticism on code, design, tests, and behavior.
- Accept maintainers' decisions about project scope and direction.

Unacceptable behavior includes harassment, personal attacks, discriminatory
language, publishing private information without permission, and sustained
disruption of project work.

Report conduct concerns privately to the repository owner. Maintainers may remove
comments, close issues, block accounts, or take other reasonable action to keep
the project usable.
