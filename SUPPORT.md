# Support

Use GitHub Issues for reproducible bugs, API questions, data-correctness reports,
and performance regressions.

Security issues should be reported privately as described in `SECURITY.md`.

Supported versions are:

- The latest tagged release.
- The active `main` branch until the next tagged release.

The minimum supported Go version is Go 1.20. CI also verifies Go 1.21 through Go
1.26.
